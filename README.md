# Fortnite Season Archive & Live Shop

A web-based Fortnite resource that combines a nostalgic look at past Battle Passes with modern game data. This project originally focused on showcasing **Season Pass characters** (such as Seasons 5 and 6) and has been updated to include a dynamic **Daily Shop** tracker.

## 🚀 Features
* **Live Daily Shop:** A newly added section that tracks the current in-game item rotation and provides a countdown to the next shop refresh.
* **Seasonal Archives:** Dedicated pages for classic Battle Passes, documenting the skins and characters that defined earlier chapters of the game.
* **Official Resources:** Quick access to Epic Games' official news, social media, and community channels.
* **Responsive Layout:** A clean interface designed to work across different screen sizes.

## 🛠️ Technologies Used
* **Frontend:** HTML5 and CSS3 (W3C Validated).
* **JavaScript:** Powering the real-time shop countdown and data fetching.
* **Deployment:** Hosted via GitHub Pages.

## 📜 Project History
This site began as a static project focused on archiving seasonal Fortnite content. It has since been expanded to include live features, bridging the gap between historical game data and current in-game events.
